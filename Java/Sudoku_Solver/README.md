# Sudoku Solver
This project is to solve any sudoku puzzle input into a [9][9] array. 0s represent empty spots.

Example input (Array Format):

{
  {0,1,8,0,0,0,0,2,6},
  {0,0,0,0,0,0,3,0,0},
	{0,0,3,1,5,0,0,0,0},
	{0,0,0,9,0,0,4,0,3},
	{0,5,0,0,4,0,8,0,7},
	{0,0,0,0,0,0,0,0,0},
	{8,0,0,0,3,0,0,0,0},
	{0,9,5,0,0,7,0,0,0},
	{4,0,0,6,0,2,7,9,0}
}
